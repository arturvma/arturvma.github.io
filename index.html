Esta é uma página que lê textos em voz alta. Isso é feito de graça, através de um programa de sintetizador de fala que fala dizendo as palavras e frases de qualquer escrita inserida. Esta página pode ser usada como ditador, simulador de alto-falante ou simplesmente como narrador virtual ou reprodutor de texto.

Digite o texto completo a ser lido na área de texto principal. Você também pode inserir o endereço de uma página da Web cujo texto você deseja ler. Em seguida, pressione o botão Ler para começar a ler; o botão Pausar interrompe a leitura para continuar quando o botão Ler é pressionado novamente. Cancelar para de ler, deixando o aplicativo pronto para iniciar novamente. Limpar remove o texto digitado, deixando a área pronta para uma nova entrada. O menu suspenso permite selecionar o idioma da voz com a qual a leitura é feita e, em alguns casos, o país de origem. Essas vozes são naturais, algumas masculinas e outras femininas.
